Island

An island is, very simply, just a way to set an element apart with some extra padding.

Look at me, I’m on an island!

Modifiers

Use the is-bordered class to add a rounded border:

Now I’m easier to see.

Use the is-small class for less padding:

Look at me, I’m on an island!

Use the is-large class for more padding:

Look at me, I’m on an island!

Use the is-extra-large class for a lot more padding:

Look at me, I’m on an island!
